# Goal
-Web app that connects riders (who arrived at airport) and drivers. Unlike uber which use cars. This app will use 16 passanger vans to transport clients
therefore multiple clients in a vehicle.

## user case

#### User
- as a user I want to book a ride
- as a user I want to see price of ride
- as a user I track car as it comes up
- as a user I want to review ride after complete
- as a user I want to see rating before booking

#### Driver
  - as a driver I want to be able to sign in to allow me to recevive trips
  - as a driver I want to sign off
  - as a driver I want to see where to drive to
  - as a drive, i want to see where riders are going
